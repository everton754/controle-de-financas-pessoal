# Controle de finanças pessoal

## ✨Sobre o projeto

O projeto de controle de finanças pessoal é a forma mais básica de cuidar do seu dinheiro, administrando bem suas finanças, você estará ciente das suas receitas, despesas e entre outros aspectos.

<br>

## 🚴‍♂️Passo a Passo

Para rodar localmente, primeiro rode o comando:
> ```yarn``` para instalar as dependências do projeto.
>  

Em seguida:
> ```yarn start``` para rodar a aplicação.

<br>
